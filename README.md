# Academic Registration System

## Description
This project is a basic academic registration system that allows users (students) to register and manage their academic information. It simulates how a real university registration system works.

## Technologies Used
- HTML
- CSS
- Javascript
- PHP
- MySQL

## Features
- Student registration form
- Data capture and storage in a database
- Input validation
- Basic user interaction
- Display of registered student information

## Purpose of the Project
This project was developed to understand how web applications interact with databases and to practice building functional systems used in real-world environments.

## What I Learned
- Connecting front-end to back-end using PHP
- Working with MySQL databases
- Handling user input and validation
- Building structured web applications
